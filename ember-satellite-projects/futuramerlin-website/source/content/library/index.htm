Using the Web environment

The Media Library includes a Web environment as a Vagrant virtual machine. For more information on installing and using Vagrant, visit the Vagrant Documentation Web site at https://docs.vagrantup.com/ (HashiCorp n.d.) or reference the book Vagrant: Up and Running (Hashimoto, Mitchell, 2013) (Hashimoto 28 May 2013). Vagrant requires VirtualBox to be installed. When using it, it may be advisable to disconnect the media library’s storage medium before running vagrant destroy, because if you are running Vagrant as a user with write permission to the folder, it may follow the symlink and erase it. This seems to possibly be a bug in Vagrant; this may be related to this issue https://github.com/MSOpenTech/vagrant-azure/issues/30 (npoggi 2014)

To start using the Web environment, cd to the “Meta/Web environment” folder. If there is not a Vagrantfile in the folder (there should be), run:

vagrant init https://archive.org/download/virtualbox_201501/virtualbox.box

and update the Vagrantfile as desired. If there is already a Vagrantfile, continue to the next step:

vagrant up
